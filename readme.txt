A WordPress plugin to disable selected RSS feeds and author archives with options in the admin panel.

== Description ==
SmartUpWorld Feed Control helps WordPress admins easily disable RSS feeds and author archives for SEO and privacy reasons. Includes settings and status pages.

== Installation ==
1. Upload the `smartupworld-feed-control` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
